# アプリケーション概要
タスクの管理を行う。進行状況に応じて未完了、進行中、完了済みにステータスを変更。

# URL
https://task-app111.herokuapp.com/

# 機能一覧
**ユーザー機能**
  
  ・新規作成
  
  ・編集
  
  ・削除
  
  ・ログイン
  
  ・ログアウト
  
  ・ログイン情報の保持

**タスク管理機能**
  
   ・タスクの新規作成
  
  ・削除
  
  ・ステータスを未完了、進行中、完了済みに変更
  
  ・検索
